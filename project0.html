  1. Project 1 - Pac-Man Underwater (Python Turtle)
    Wrote a Python program called Project1.py that uses the Turle graphics library to draw an underwater Pac-Man scene with a sky, ocean, sun, seafloor, shore pole, Pac-Man, and red power pellets.     Created a winter scene in Project2.py using Turtle graphics and decomposition into functions like draw_rectangle and draw_circle. This project helped me practice breaking a larger drawing into reusable functions and organizing my code.
  3. Project 3 - Refactored Winter Scene with Modular Functions (Python Turtle)

I also experimented with an enhanced scene that adds multiple houses, trees, snowpeople, and extra snowflakes. In this project, I focused on code organization, modular design, and readability.
